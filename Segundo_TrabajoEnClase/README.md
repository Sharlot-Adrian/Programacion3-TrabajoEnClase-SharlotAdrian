Descripcion:

Ejercicio realizado en clase. 

Desarrolle un programa en Java que permita registrar la información de 5 visitantes utilizando un arreglo.
El programa debe solicitar por teclado los siguientes datos de cada visitante: Nombre, Cédula, Edad y Luego de registrar los 5 visitantes, el sistema debe mostrar en pantalla la información almacenada de todos los visitantes.

Requisitos del ejercicio: Utilizar arreglos (Luego se solicitó cambiarlos por ArrayList), utilizar ciclos for utilizar entrada de datos con Scanner y mostrar los datos organizados en consola.
